# Images Directory

Please place your profile photo here with one of these filenames:
- `profile-photo.png`
- `profile-photo.jpg`
- `profile-photo.jpeg`

The photo will be displayed in:
1. Hero section (circular, 200px)
2. About Me section (rounded rectangle, 280px)

Recommended image specifications:
- Format: PNG or JPG
- Size: At least 400x400 pixels
- Aspect ratio: Square (1:1) works best
- File size: Optimized for web (under 500KB recommended)
